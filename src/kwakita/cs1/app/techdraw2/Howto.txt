# クラス階層について調査したくなったときは？

興味のあるクラスを選択し，コンテキストメニュー（対象物を右クリックしたときに表示されるメニューのこと）から "Quick Type Hierarchy"，あるいは "Open Type Hierarchy" を選択する．前者は浮遊ウィンドウに，後者は新しいPaneにクラス階層を表示する．

# あれを探したい

## いま見ているコードに出現したクラスの定義を簡単に見るには？

クラス名のコンテキストメニューから "Declarations > Project" を選択．

なお declaration とは宣言のこと．この操作は，現在，眺めているプロジェクトのなかから，指定したクラスの宣言を探すことを指示している．

## あるクラスを宣言を変更する前に，そのクラスが利用されているコードを探したい

クラス名のコンテキストメニューから "Rerefences > Project" を選択．

なお reference とは参照のこと．この操作は，現在，眺めているプロジェクトのなかから，指定したクラスを利用している箇所（したがって，当該クラスを参照している箇所）を探すことを指示している．
